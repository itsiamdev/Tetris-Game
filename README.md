# 🎮 Tetris Game

Welcome to the classic **Tetris Game**! 🧩 This is a web-based implementation of the timeless puzzle game where you arrange falling blocks to clear lines and achieve high scores.

## ✨ Features

- 🎯 **Classic Gameplay**: Experience the original Tetris mechanics with falling tetrominoes.
- 🎨 **Colorful Blocks**: Each shape has its own vibrant color for easy identification.
- 📊 **Score Tracking**: Keep track of your level, lines cleared, score, and time.
- 🔮 **Next Shape Preview**: See the upcoming piece to plan your moves ahead.
- ⏸️ **Pause Functionality**: Press Esc to pause and resume the game anytime.
- 📱 **Responsive Design**: Play on your desktop or mobile device.

## 🎲 How to Play

1. Click the **Start** button to begin the game.
2. Use the **arrow keys** to move and rotate the falling pieces:
   - ⬅️ **Left Arrow**: Move left
   - ➡️ **Right Arrow**: Move right
   - ⬇️ **Down Arrow**: Move down faster
   - ⬆️ **Up Arrow**: Rotate piece
3. Complete horizontal lines to clear them and earn points.
4. The game speeds up as you level up! 🚀
5. Press **Esc** to pause/unpause.

## 🛠️ Technologies Used

- 🌐 **HTML5**: Structure and canvas for the game board
- 🎨 **CSS3**: Styling and animations
- ⚙️ **Vanilla JavaScript**: Game logic and interactivity
